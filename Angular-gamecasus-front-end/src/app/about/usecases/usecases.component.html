Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

BioscoopCasus

Het project houdt als volgt in. Ik moet een angular applicatie maken waarbij ik een bepaald onderwerp toezeg. Ik heb bij dit project gekozen voor een BioscoopCasus. Hierbij maak ik gebruik van 4 enititeiten namelijk: Moviescreening, Halls, Movie en natuurlijk de User.

  • Moviescreening wordt gebruikt, omdat bij een flim vaak een bepaalde plek wordt gekozen samen met een gekozen film. Samen wordt dat een Moviescreening
  • Halls wordt gebruikt, omdat je bij een film kijken vaak een plek moet hebben om het te bekijken.
  • Movie wordt gebruikt, omdat je naar de biocoop gaat om een film te zien.
  • User wordt gebruikt, omdat je hiermee kun authenticeren en een film kan bekijken.
